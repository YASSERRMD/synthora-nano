# Changelog

## [0.1.0] - 2025-01-10

### Added
- Initial project setup with Vite, React, and TypeScript
- Design system with navy/gold theme and dark mode
- IndexedDB data layer with 9 domain schemas
- Document parsers for PDF, Markdown, HTML, and plain text
- Chrome built-in AI runtime integration
- Structured paper analysis pipeline
- Research library with search and filtering
- Cross-paper comparison engine
- Concept linking and knowledge graph
- Grounded research assistant with citations
- Workspace export/import with checksums
- PWA with offline support
- Security hardening with CSP and content validation
- 207 unit tests
